## 📖 核心宣言 (Manifesto)

**这不是一个简单的工具库，而是一个对抗信息熵增的“拉普拉斯妖”。**

渗透测试的本质是信息博弈。然而，我们面临的现状是：
1.  **工具的高熵**: 脚本散落在 GitHub 各个角落，配置复杂，互不兼容。
2.  **思维的高熵**: 方法论停留在攻击者的脑海中，缺乏标准化的 Prompt 和 SOP。
3.  **知识的高熵**: 优质的落地成果淹没在 CSDN、公众号和碎片化的文章中。

本仓库致力于**维护这三个方向的“负熵”**，将它们收录、清洗、结构化。

### 🔗 支柱三：私有/小众武库 (Niche Arsenal)
> **拒绝平庸。我们不收录人尽皆知的知名工具（如 SQLMap），只收录那些解决特定痛点的“冷门遗珠”与“自研魔改”。**

这里是负熵体系的“特种作战”区域。我们专注于收集那些**能显著降低特定环节熵值**，却鲜为人知的代码片段。
